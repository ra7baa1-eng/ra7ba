الخطوة 1️⃣: جهز GitHub
────────────────────────────────────
1. افتح PowerShell في مجلد المشروع
2. اكتب هذه الأوامر واحد واحد:

   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   git init
   git add .
   git commit -m "Ra7ba Platform"
   git remote add origin https://github.com/USERNAME/REPO.git
   git push -u origin main

✅ المشروع الآن على GitHub!


الخطوة 2️⃣: سجل في Render
────────────────────────────────────
1. اذهب: https://render.com
2. اضغط "Get Started"
3. اختر "Sign in with GitHub"
4. وافق على الأذونات

✅ حسابك جاهز!


الخطوة 3️⃣: أنشئ قاعدة البيانات
────────────────────────────────────
1. اضغط "New +" → "PostgreSQL"
2. املأ:
   Name: ra7ba-db
   Database: ra7ba
   Plan: Free
3. اضغط "Create Database"

✅ قاعدة البيانات جاهزة!
📝 احفظ "Internal Database URL"!


الخطوة 4️⃣: انشر Backend
────────────────────────────────────
1. اضغط "New +" → "Web Service"
2. اختر مستودع "ra7ba"
3. Root Directory: backend
4. Build: npm install && npx prisma generate && npm run build
5. Start: npx prisma migrate deploy && npm run start:prod
6. أضف Environment Variables (انظر نشر_راندر_سريع.txt)

✅ Backend شغال!


الخطوة 5️⃣: شغّل البيانات الأولية
────────────────────────────────────
1. في Backend Service → اضغط "Shell"
2. انتظر حتى يفتح Terminal
3. اكتب:

   npx prisma db seed

4. اضغط Enter

✅ قاعدة البيانات معبأة!


الخطوة 6️⃣: انشر Frontend
────────────────────────────────────
1. اضغط "New +" → "Web Service"
2. اختر مستودع "ra7ba"
3. Root Directory: frontend
4. Build: npm install && npm run build
5. Start: npm run start
6. أضف Environment Variables

✅ Frontend شغال!


الخطوة 7️⃣: احصل على الروابط
────────────────────────────────────
Backend URL: https://ra7ba-backend.onrender.com
Frontend URL: https://ra7ba.onrender.com

✅ المنصة شغالة الآن!


═══════════════════════════════════════════════════════════
          🎯 جرب المنصة
═══════════════════════════════════════════════════════════

1. افتح Frontend URL
2. اضغط "إنشاء متجر مجاني"
3. سجل كتاجر جديد

للدخول كـ Admin:
   البريد: admin@ra7ba.com
   كلمة المرور: Admin123!

✅ تمام! المنصة تعمل!
