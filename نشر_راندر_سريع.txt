الخطوة 1️⃣: ارفع المشروع على GitHub
────────────────────────────────────────────────

أ) إذا عندك Git:
   
   cd C:\Users\arinas\Desktop\saas\rahba
   git init
   git add .
   git commit -m "Ra7ba Platform"
   git remote add origin https://github.com/USERNAME/ra7ba.git
   git push -u origin main


ب) إذا ما عندكش Git - استخدم GitHub Desktop:

   1. حمّل: https://desktop.github.com/
   2. File → Add Local Repository
   3. اختر: C:\Users\arinas\Desktop\saas\rahba
   4. Publish Repository

✅ تمام!


الخطوة 2️⃣: سجل في Render
────────────────────────────────────────────────

1. اذهب: https://render.com
2. اضغط "Get Started"
3. اختر "Sign in with GitHub"
4. وافق على الأذونات

✅ حسابك جاهز!


الخطوة 3️⃣: أنشئ قاعدة بيانات
────────────────────────────────────────────────

1. في Render، اضغط "New +" → "PostgreSQL"
2. املأ:
   Name: ra7ba-db
   Database: ra7ba
   Region: Frankfurt
   Plan: Free

3. اضغط "Create Database"
4. انتظر حتى تصبح "Available"

✅ قاعدة البيانات جاهزة!

📝 احفظ "Internal Database URL" - ستحتاجه!


الخطوة 4️⃣: انشر Backend
────────────────────────────────────────────────

1. اضغط "New +" → "Web Service"
2. اختر مستودع "ra7ba"
3. املأ:

   Name: ra7ba-backend
   Region: Frankfurt
   Root Directory: backend
   
   Build Command:
   npm install && npx prisma generate && npm run build
   
   Start Command:
   npx prisma migrate deploy && npm run start:prod

4. أضف Environment Variables:

   NODE_ENV=production
   PORT=3001
   DATABASE_URL=<الصق Internal Database URL>
   JWT_SECRET=ra7ba-jwt-secret-2024-change-me
   JWT_REFRESH_SECRET=ra7ba-refresh-secret-2024-change-me
   ADMIN_EMAIL=admin@ra7ba.com
   ADMIN_PASSWORD=Admin123!ChangeMe

5. اضغط "Create Web Service"
6. انتظر البناء... (5-7 دقائق)

✅ Backend شغال!


الخطوة 5️⃣: شغّل البيانات الأولية
────────────────────────────────────────────────

1. في Backend Service → اضغط "Shell"
2. انتظر حتى يفتح Terminal
3. اكتب:

   npx prisma db seed

4. اضغط Enter

✅ قاعدة البيانات معبأة بـ:
   - 58 ولاية جزائرية
   - حساب Admin
   - إعدادات افتراضية


الخطوة 6️⃣: انشر Frontend
────────────────────────────────────────────────

1. اضغط "New +" → "Web Service"
2. اختر مستودع "ra7ba"
3. املأ:

   Name: ra7ba
   Region: Frankfurt
   Root Directory: frontend
   
   Build Command:
   npm install && npm run build
   
   Start Command:
   npm run start

4. أضف Environment Variables:

   NODE_ENV=production
   NEXT_PUBLIC_API_URL=https://ra7ba-backend.onrender.com/api
   NEXT_PUBLIC_APP_NAME=Ra7ba

   ⚠️ استبدل "ra7ba-backend" باسم Backend Service!

5. اضغط "Create Web Service"
6. انتظر البناء... (3-5 دقائق)

✅ Frontend شغال!


الخطوة 7️⃣: حدّث CORS
────────────────────────────────────────────────

1. ارجع لـ Backend Service
2. اذهب لـ Environment
3. أضف:

   FRONTEND_URL=https://ra7ba.onrender.com

   ⚠️ استبدل "ra7ba" باسم Frontend Service!

4. اضغط "Save Changes"

✅ CORS محدّث!

          🐛 حل المشاكل
═══════════════════════════════════════════════════════════

❌ Backend فشل في البناء؟
   → تأكد من Build Command صحيح
   → npm install && npx prisma generate && npm run build

❌ Migration فشل؟
   → في Shell: npx prisma migrate deploy --force

❌ Frontend لا يتصل بـ Backend؟
   → تأكد من NEXT_PUBLIC_API_URL صحيح
   → تأكد من FRONTEND_URL في Backend صحيح

❌ Database Connection Error؟
   → استخدم Internal Database URL (وليس External)
